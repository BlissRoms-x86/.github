# BlissOS legacy
Welcome to BlissOS, this place is the beginning of our journey & a part of our legacy. It's the home of BlissOS sources from 15 and below. 

If you are looking for newer sources, check out the new organization here: https://github.com/BlissOS

# Downloads
If you are looking for download links, go to our website: https://blissos.org/

# Sources & Documentation
Currently there are 2 sources that are still actively supported in this organization and verified to be buildable. 

[BlissOS 14 source (based on Android 11)](https://github.com/BlissRoms-x86/manifest/tree/r11-x86) <br>
[BlissOS 15 source (based on Android 12L)](https://github.com/BlissRoms-x86/manifest/tree/arcadia-x86)

Documentation to build the OS is already in the source's README.md. If you are looking for a guide to install & configure BlissOS, check https://docs.blissos.org/
